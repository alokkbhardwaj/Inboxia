# Inboxia
A secure, self-hosted email platform built for learning system design

## Features
- SMTP-based email sending
- IMAP-based email receiving
- Webmail interface
- Secure email delivery using industry standards

## Tech Stack (Planned)
- Postfix (SMTP)
- Dovecot (IMAP)
- Roundcube (Webmail)
- Docker
- Linux (Ubuntu)

## Goal
To build a secure, production-style email system for learning and resume enhancement.
